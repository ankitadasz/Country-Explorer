# 🌍 Country Explorer

Country Explorer is a modern and responsive web application built with React.js that allows users to explore countries from around the world. The application fetches real-time country data from a REST API and provides features such as country search, region-based filtering, alphabetical sorting, and detailed information for each country. Built with React Router for seamless navigation and Axios for API requests, the application offers a clean and intuitive user experience across desktop and mobile devices.

## ✨ Features

- 🌍 Browse countries from around the world
- 🔍 Search countries by name
- 🌎 Filter countries by region
- 🔤 Sort countries alphabetically (A–Z / Z–A)
- 📄 View detailed information for each country
- 📱 Fully responsive design
- ⚡ Fetch real-time data using Axios and REST API
- 🛣️ Client-side routing with React Router
- ⏳ Loading indicator while fetching data
